# Purchase Intent Classifier

## Intro
* The notebook, main.ipynb, contains the report as well as the code for the application.

## How to view
* Open the notebook and go through it in order to view the application demo, review the evaluation results and read the analyses.
* You do not have to open any of the other files. Everything is viewable from the notebook.
* The notebook can be reviewed without rerunning. I've saved the notebook with my results so you don't actually have to run it again.
* DO NOT RUN THE NOTEBOOK AGAIN. IT WILL COST YOU 30 MINUTES TO COMPLETE. You will also need to set up your own environment as well as have your own OpenAI API key to do so.
